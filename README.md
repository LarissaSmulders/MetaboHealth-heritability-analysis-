# A repository for the MetaboHealth heritability analysis

This repository contains the code written in `R` to analyze the MetaboHealth score data published in the Journal of GeroScience by Smulders _et al._: _The MetaboHealth score is 40% heritable and is influenced by frailty status, BMI, and smoking in Swedish twins_ (DOI:10.1007/s11357-026-02223-w)
